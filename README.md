### 使用教程：

#### 方法1：直接运行编译好的包(适用于普通用户)：

1：根据你的系统，从[Releases · itsHenry35/ledu (github.com)](https://github.com/itsHenry35/ledu/releases)下载程序

2：运行程序

#### 方法2：从源码运行：

1：提前安装Python

2：`git clone https://github.com/itsHenry35/ledu.git`

3：`cd ledu`

4：`pip install -r requirements.txt`

5：`python main.py`
